command line:


open.
Opens folder in finder

mkdir folderName
Make a folder (make directory, eg. mkdir dataVis)

touch fileName.extention
Creates new file with given extention(eg. touch index.html)

clear
Clears the command line

git add -A or git add --all
Adds all changed files to the commit

git commit -m "message"
Commits changes to push que

git push
Pushes all changes to branch

git checkout -b branchName
Creates a new branch with name (eg. git checkout -b jessesBranch)

git branch
Shows existing branches (if there are none it will show nothing)

git branch branchName
Activates the branch as the working file (eg. git branch jesse (will now be working off the branch named 'jesse').


git merge branchName
Will merge the branch with the master (you must be on the master file and have already commited the branch changes. eg. git branch jesse > work on the file > git commit -m "message" > git push (pushes the changes to the branch) > git branch master > git merge jesse (will merge the changes of the jesse branch with the master.)
